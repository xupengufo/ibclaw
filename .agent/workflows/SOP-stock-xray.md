# SOP: 个股深度诊断 (Stock X-Ray)

当用户询问针对某一特定股票（如 AAPL、NVDA）的深度分析、买卖建议或整体评估时，你**必须严格执行以下串行流程**，不允许偷懒或只凭借大模型的先验知识回答。

## 阶段 1：数据采集 (Data Gathering)

请按顺序调用以下工具收集这只股票的多维度数据：

1. **基本面与护城河**：
   运行 `./ibkr fundamentals <SYMBOL> --json`
2. **当前走势与技术压力**：
   运行 `./ibkr analyze <SYMBOL> --json`
3. **华尔街机构看法**：
   运行 `./ibkr ratings <SYMBOL> --json` (只看前5-10条近期的)
4. **内部高管动向**：
   运行 `./ibkr insider <SYMBOL> --json`

## 阶段 2：数据交叉验证 (Cross-Validation)

在向用户输出结果前，请在你的思维链中核对以下共振：
- 价值回归信号：基本面的 `P/E` 是否同行业偏低？技术面上是否刚好出现 `Oversold` (超卖)的 RSI 信号？
- 主力行为一致性：内部高管是否在近期有大额的 `Buy` 买入行为？同时各大投行是否正在上调评级 (\`Upgrades\`)？

## 阶段 3：最终报告输出 (Report Delivery)

请合并以上数据，以一个**顶级华尔街分析师**的口吻按照以下 Markdown 格式结构化地给用户回复。**杜绝空话，全部用刚才采集到的实质数据填充**。

```markdown
# 🏥 <SYMBOL> 深度诊断报告

## 1. 核心体检单 (Snapshot)
- P/E (市盈率) / PEG: [基本面数据]
- RSI (超买超卖): [技术面分析结果]
- 52周均线乖离: [技术面分析结果]

## 2. 筹码与内部人 (Smart Money)
[总结内部交易行为，比如“过去 1 个月内出现 3 笔高管抛售，总计 $XXX” 或 “近期无明显的内部人异动”。]

## 3. 华尔街共识 (Wall Street Consensus)
[给出分析师的平均推荐偏向，以及近期更新过的评级和目标价情况。]

## 4. 交易层建议 (Actionable Advice)
- 支撑位: [提炼技术分析中的下沿支撑]
- 压力位: [提炼技术分析中的上沿阻力]
- 阶段性定调: [ 综合上述 1、2、3 给出明确判断：当前属于强势突破/超跌企稳/见顶风险积聚/垃圾股坚决规避 ]
```
