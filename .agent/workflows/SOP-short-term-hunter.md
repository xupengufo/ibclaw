# SOP: 短线异动海选 (Short-term Hunter)

当用户并未指定具体的股票，而是要求你利用工具寻找短线、日内或波段机会时，请以**动量（Momentum）和极值反转（Mean Reversion）**为核心逻辑，分两个步骤自动执行筛选，然后精炼 1-2 支股票汇报给用户。

## 步骤 1：盘面情绪海选 (Market Scanner)

首先必须弄清楚资金当下在往哪些方向涌动。你需要跑一个 IBKR 的实时市场扫雷器（粗筛）：

```bash
# 获取当前整个市场成交量/换手率异动最厉害的 15 支标的
./ibkr scanner --code HOT_BY_VOLUME --size 15 --json
```

## 步骤 2：基本面/技术面过滤 (Finviz Screener)

对第 1 步里拿到的一堆代码，利用 Finviz 多维选股做高斯过滤，剔除掉那些没有基本面支撑的纯概念游资炒作股。

```bash
# 从刚才提取的代码里挑 2-3 只顺眼的标的（比如属于主线科技或医疗的），跑深度体检：
./ibkr fundamentals <SYMBOL> --json
./ibkr analyze <SYMBOL> --json
```

*(如果用户明确要求找超跌反弹的，你就不要做步骤 1，而是直接调用 Finviz 技术面选股：)*
```bash
./ibkr screen --signal Oversold --size 5 --json
# 获得这批股票后再单独测它们的基本面。
```

## 步骤 3：精选呈现 (Final Picks)

禁止像倒流水账一样把 API 返回的全集贴给用户。作为交易副驾，你的职责是做决策。最终给用户的回复**只能推荐不超过 2 只**你认为逻辑最自洽的标的。

**输出模板格式：**

```markdown
# 🎯 今日短线雷达精选

经过两层交叉验证，当前最值得留意的异动短线标底如下：

## 1. <SYMBOL> (<公司名>)
- **入选逻辑**: (比如：“由于期权异动量暴增，且基本面 P/E 处于行业低位...”)
- **技术形态**: (提炼技术指标的形态和支撑阻力)
- **风险提示**: (如: "财报即将在 3 天后发布" 或 "高管存在抛售行为")

## 2. <SYMBOL> (<公司名>)
- ...

> *注：标的由 `ibclaw` 基于 [HOT_BY_VOLUME/超卖 信号] 自动化计算筛选。交易前请关注大盘日内风向。*
```
